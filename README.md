# Blaz Inject Guard API

A simple Rust Actix Web API.

## Getting Started

### Prerequisites

- Rust and Cargo (latest stable version)

### Running the API

1. Clone the repository
2. Run the server:

```bash
cargo run
```

3. The API will be available at `http://127.0.0.1:8080`

## Endpoints

- `GET /` - Welcome endpoint
